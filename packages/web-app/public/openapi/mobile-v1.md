# Recuring Mobile API v1

REST API for the Recuring field-operator mobile application.

Authentication — pass the session token from `POST /auth/login` as `Authorization: Bearer <token>`.

CSRF protection — all mutation endpoints (POST / PATCH / DELETE) require the `X-CSRF-Token` header obtained from `GET /auth/me`.

Rate limits — mobile org bucket is 10× the standard org bucket.

Spec endpoint — `GET /api/mobile/v1/openapi.json` (no auth required).

## Auth

| Method | Path | Summary |
|--------|------|---------|
| `POST` | `/auth/login` | Login with phone or employee_id + password |
| `POST` | `/auth/logout` | Invalidate the current session |
| `GET` | `/auth/me` | Return current session user and a fresh CSRF token |
| `POST` | `/auth/otp/request` | Send a 6-digit OTP to the given phone number |
| `POST` | `/auth/otp/verify` | Verify OTP and return a session token |

## Jobs

| Method | Path | Summary |
|--------|------|---------|
| `GET` | `/jobs` | List jobs assigned to the authenticated technician |
| `GET` | `/jobs/{id}` | Get a single job with customer and asset detail |
| `POST` | `/jobs/{id}/status` | Transition job status (en_route → arrived → in_progress → completed) |
| `POST` | `/jobs/{id}/fail` | Cancel / fail a job with a mandatory reason |
| `POST` | `/jobs/{id}/reschedule` | Reschedule a job to a different date |
| `POST` | `/jobs/{id}/notes` | Update the service report / work notes for a job |
| `GET` | `/jobs/{id}/proofs` | List all proof photos for a job |
| `POST` | `/jobs/{id}/proofs` | Attach a proof photo to a job |
| `DELETE` | `/jobs/{id}/proofs/{proofId}` | Delete a proof photo from a job |

## Complaints

| Method | Path | Summary |
|--------|------|---------|
| `GET` | `/complaints` | List complaints assigned to the authenticated technician |
| `GET` | `/complaints/{id}` | Get a single complaint with full timeline, customer, and asset detail |
| `POST` | `/complaints/{id}/status` | Transition a complaint status (in_progress, on_hold, resolved) |
| `POST` | `/complaints/{id}/jobs` | Create a service job from an existing complaint |
